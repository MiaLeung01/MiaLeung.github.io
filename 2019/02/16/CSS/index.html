  1. CSS 盒模型
    dom 元素是由四部分组成，从内到外为 content、padding、 border 、margin
    当你用 CSS 给元素设置 width 和 height 的时候，只会应用到 content 区，如果还有 padding 和 border ，
    浏览器绘制的时候，标准的盒子宽度和高度会加上 padding 和 border。

  2. box-sizing
    content-box: 默认值，标准盒子模型。 width 与 height 只包括内容的宽和高， 不包括边框（border），内边距（padding），外边距（margin）。
    border-box: width 和 height 属性包括内容，内边距和边框，但不包括外边距。

  3. BFC (Block Fromatting Context) https://www.cnblogs.com/libin-1/p/7098468.html
    作用：

    1. 阻止 margin 融合（外边距折叠）：处于不同BFC中的元素外边距不折叠
    2. 包含浮动元素
    3. 避免文字环绕 ：BFC 内子元素贴紧父元素左外边缘

    生成 BFC：

    1. position 不是 static 或者 relative
    2. float 不为 none
    3. overflow 不是 visible
    4. display 的值为 inline-block、 table-cell、 table-caption、 flex 或者 inline-flex

  4. 层叠上下文 Stacking Context https://www.jianshu.com/p/0f88946a0746
    HTML中的三维概念，每个盒模型的位置都是三维的，分别是平面画布的 x 轴、 y 轴 和层叠的 z 轴。
    产生：

    1. HTML中的根元素本身就具有层叠上下文，称为“根层叠上下文”。
    2. 普通元素设置position属性为非static值并设置z-index属性为具体数值，产生层叠上下文。
    3. CSS3中的新属性也可以产生层叠上下文。
      层叠顺序：
      background/border
      z-index<0
      block 块级水平盒子
      float 浮动盒子
      inline/inline-block 水平盒子
  z-index auto/0
    z-index > 0

  5. 居中布局
    （1）水平居中

    • 行内元素：text-aligin: center
    • 块级元素：margin: 0 auto

    • absolute + transform

    • flex : justify-content: center
      （2）垂直居中

    • line-height
    • absolute + transform
    • flex + align-items: center
    • table
      (3)水平垂直居中
    • absolute + transform
    • flex + justify-content + align-items
